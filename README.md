# Grow Boostly

Official website for **Grow Boostly** — an AI-powered lead generation and digital marketing agency. PHP-based marketing site with service pages, industry landing pages, pricing, SEO schema, and conversion-focused layouts.

**Live site:** [https://www.growboostly.com](https://www.growboostly.com)

## Tech stack

- PHP (includes-based pages)
- Bootstrap 5, jQuery, Swiper, Slick, GSAP
- SEO: meta tags, Open Graph, JSON-LD schema

## Local development

```bash
php -S localhost:8000
```

Then open [http://localhost:8000](http://localhost:8000)

## Project structure

| Path | Description |
|------|-------------|
| `index.php` | Homepage |
| `services/` | Service landing pages |
| `industries/` | Industry landing pages |
| `pricing/` | Pricing pages |
| `includes/` | Shared PHP includes |
| `header.php` / `footer.php` | Site header and footer |
| `assets/` | CSS, JS, images, fonts |
